# Regras de UI e Estrutura de Tabelas
- As tabelas devem ter TODAS as suas colunas renderizadas na tela desde o Passo 1 (início).
- A tabela funciona melhor se estiver estruturalmente completa no HTML. As colunas são preenchidas com dados a cada passo.
- Se uma coluna não for necessária visualmente em um dado momento, prefira ocultá-la via CSS (display: none) ao invés de removê-la do DOM.
